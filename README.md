# Titanic-Survival-Prediction-Web-App 🚢
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

It is built using Python, Scikit-learn, and Streamlit and deployed as an interactive web application.

---

## 📌 Project Overview

The goal of this project is to build a machine learning model that predicts passenger survival based on features such as:

* Passenger Class
* Sex
* Age
* Number of Siblings/Spouses aboard
* Number of Parents/Children aboard
* Fare

The model is trained using the famous Titanic dataset.

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/titanic-survival-prediction.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Streamlit app

```
streamlit run streamlit_app.py
```

4. Open the browser

```
http://localhost:8501
```
